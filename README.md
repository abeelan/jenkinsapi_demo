# 安装
```shell script
$ pip install jenkinsapi
```

# 过程记录

jenkins 实例化时，密码最好填写生成的 api token

生成方法：
- jenkins 点击头像，进入设置
- 点击 Add New Token
- 将生成的 api token 填入密码参数内
